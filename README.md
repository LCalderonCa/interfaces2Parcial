# ReservaPlus - Sistema de Gestión de Reservas

## Levantar el Proyecto

```bash
npm install
npm start
```
Abre: http://localhost:3000

## Credenciales Demo
- Admin: admin@reservaplus.com / admin123
- Cliente: carlos@mail.com / cliente123

## Rutas
- / → Página pública
- /login → Login
- /registro → Registro
- /reservar → Reservar mesa (requiere login)
- /mis-reservas → Ver mis reservas (requiere login)
- /admin → Panel admin (requiere rol admin)
- /admin/reservas → Gestión de reservas
- /admin/mesas → Gestión de mesas
- /admin/calendario → Calendario
- /admin/reportes → Reportes
